# Social Media

Organic social-media growth skills — parallel to PLG-Architect and the SEO suite. One skill per platform, each built to grow a *real, revenue-tied audience* (not vanity followers), the sacrifice-first and evidence-based way the other suites work.

Install from the [repo root README](../README.md#quick-install).

## Skills

**Built**
1. `x-growth` — grow an audience on X (Twitter) that ties to revenue: reframe follower goals into the real outcome, reality-check whether your ICP is even on X, pick one sustainable content lane, design the X-native growth loop and the profile→link→capture funnel, then hand off.

**Planned** (parallel platform skills, same pattern)
- `social-context` — optional gatekeeper: inherit founder-context, pick the platform(s) worth being on, route to the right skill.
- `linkedin-growth`, `instagram-growth`, `tiktok-growth`, `youtube-growth`, `threads-growth`.
- `content-repurposing` — turn one idea into platform-native posts across the suite.

## Context: standalone, but inherits when it can
Each skill works on its own, but reads `.claude/founder-context.md` if PLG-Architect already wrote it, so business basics aren't re-entered. No hard dependency between suites — the coupling is just the shared `.claude/` files.

## Principles
Founder-first · audience-to-revenue · sacrifice-focused · evidence-typed · elaborate + honest · never ban-risk.
