# PLG-Architect

> Founder-level Product-Led Growth strategy skills for Claude.
> By Ranjeet Vimal.

## The 6 Skills

| Skill | What It Does |
|---|---|
| `plg-strategy` | Orchestrator — asks hard questions, red-teams, produces quantified bets |
| `icp-research` | Validates ICP through evidence, finds desperate customers |
| `competitive-intel` | Moat analysis — what can be copied in 30 days vs. 3 years |
| `viral-loops` | 1-2 quantified loop bets, not 20 tactics |
| `content-strategy` | 1-2 channel bets with explicit sacrifices |
| `growth-metrics` | Validation tools for bets, not vanity dashboards |

## Install This Suite

```bash
npx skills add ranjeeetvimal/growth-skills --suite PLG-Architect
```

## How It Works

1. **Hard questions first** — Challenges your assumptions before giving advice
2. **Product teardown** — Actually signs up and uses your product
3. **Macro thesis** — Why does your business exist NOW?
4. **Red team** — 3 ways you could fail
5. **Moat analysis** — What's actually defensible?
6. **3 bets** — Quantified, with validation plans
7. **Sacrifices** — 5+ things you're NOT doing
