# Loop Patterns + Activation

## Loop type menu (pick 1-2, not 5)
| Loop | Mechanic | Fits | Honest K |
|---|---|---|---|
| Branded-artifact | User output carries your brand to new eyes | Product makes visible/shareable output | 0.1–0.4 |
| Referral/invite | Mutual-benefit invites (credits, collab) | Social/collaborative product | 0.2–0.7 |
| Content/demo-as-funnel | The content is the live demo | Visual, instant-to-show product | Compounds via reach, not K |
| Collaboration | User must invite others to get value | Team/multiplayer product | 0.5–1.5 (strongest) |
| Marketplace/network | More users = better product | Two-sided / data network | True network effect |
| Switcher land-grab | Capture migration when incumbent stumbles | A big rival just angered its base | Time-boxed |

## K-factor math (show it or it's fiction)
K = (invites or impressions per user) × (conversion rate of those).
K < 1 = amplifier that lowers CAC, NOT exponential growth. Say this plainly —
most "viral" plans are really CAC reducers. Target realistic: K 0.15 → 0.3 is a
strong content loop. Instrument the loop input → visit → signup.

## Activation (the loop dies if users never reach value)
- **Aha = first real value delivered**, named precisely (not "signed up").
- **TTV target** < 5 min self-serve; instrument the riskiest step separately.
- **Template-first onboarding**, forced self-test ("do X now"), celebrate the aha.
- **Activation benchmark:** ≥50% of signups reach the aha within 24h is healthy.

## Free-to-paid as a bet
Name the current conversion (or "unknown"), the target with a source, the change,
the expected revenue impact, and the A/B test. Classic lever: gate the feature the
power user hits naturally; let the free tier keep advertising you.

## Sacrifices
Name the loops you are NOT building and why. Two instrumented loops beat five hopeful ones.
