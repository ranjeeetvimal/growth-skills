# PLG Frameworks & Mental Models — Reference

## 1. The PLG Flywheel
```
Acquisition → Activation → Retention → Monetization → Expansion
     ↑_________________________________________________↓
```
The product itself is the primary growth driver.

## 2. Time-to-Value (TTV) Benchmarks
| Product Type | Target TTV |
|---|---|
| Consumer apps | <1 minute |
| Simple SaaS | <3 minutes |
| Medium SaaS | <5 minutes |
| Complex B2B | <1 hour |
| Enterprise | <1 day |

## 3. The Aha Moment
The moment a user experiences core value. Examples:
- **Slack**: First message sent and replied to
- **Dropbox**: First file synced
- **Canva**: First design exported
- **Stripe**: First payment processed
- **Duolingo**: First lesson completed

Engineer it by: removing friction, celebrating the moment, making it unmissable.

## 4. Viral Coefficient (K-Factor)
```
K = (Invites per user) × (Conversion rate of invites)
```
- K > 1.0 = exponential growth
- K = 0.3-0.7 = strong engine
- K < 0.2 = needs paid boost

## 5. Pricing Psychology Models
- **Freemium**: Free forever with gates
- **Free Trial**: Time-limited full access
- **Usage-Based**: Pay for what you use
- **Seat-Based**: Per-user pricing
- **Feature-Based**: Tiered by features
- **Value-Based**: Price tied to outcome
- **Hybrid**: Combination of above

## 6. The Hook Model (Nir Eyal)
```
Trigger → Action → Variable Reward → Investment
```

## 7. Jobs-to-be-Done
| Type | Question | Example |
|---|---|---|
| Functional | What practical task? | "Automate DMs" |
| Emotional | How do they want to feel? | "In control" |
| Social | How do they want to be seen? | "Professional" |

## 8. Pirate Metrics (AARRR)
Acquisition → Activation → Retention → Revenue → Referral

## 9. Product-Qualified Lead (PQL)
A user who experienced meaningful value and is ready to buy.
Signals: hit usage thresholds, used advanced features, invited team.

## 10. Expansion Revenue Triggers
- Usage expansion (hitting limits)
- Feature expansion (trying advanced features)
- Team expansion (adding collaborators)
- Outcome expansion (success in one use case)

## 11. ICE Scoring
```
ICE = Impact (1-10) × Confidence (1-10) × Ease (1-10)
```

## 12. Growth Experiment Loop
Hypothesis → Build → Ship → Measure → Learn → Iterate

## 13. Product-Market Fit Survey (Sean Ellis)
"How would you feel if you could no longer use [product]?"
- 40%+ "very disappointed" = PMF achieved
- 25-40% = Getting close
- <25% = Need to pivot

## 14. Bowling Pin Strategy
Knock down adjacent markets one at a time.

## 15. The Cold Start Problem
Solutions: single-player value, manual curation, piggybacking, incentives, founder-led.
